# Poker Analytics Platform

A modern poker analytics application built for serious poker players.

The platform stores every hand, action, and session in a structured database, enabling advanced player analysis, probability calculations, leak detection, and long-term performance tracking.

## Features

* Hand history storage and replay
* Hand strength and equity calculations
* Monte Carlo simulations
* Player statistics (VPIP, PFR, 3Bet, CBet, WTSD, W$SD, bb/100)
* Session tracking and performance analysis
* Automatic player profiling
* Leak detection and improvement suggestions
* Interactive dashboards and reports
* AI-powered analytics assistant
* Natural language queries over poker data
* Local-first architecture with SQLite
